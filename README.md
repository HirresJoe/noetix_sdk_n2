# noetix_sdk_n2
Noetix robot N2 sdk.

### Prebuild environment
* OS  (Ubuntu 20.04 LTS or Ubuntu 22.04 LTS)  
* CPU  (aarch64 or x86_64)   
* Compiler  (gcc version 9.4.0) 

### Environment Setup

Before building or running the SDK, you can read the documentation linked here to ensure the following dependencies are installed:

[N2 SDK Environment Setup](https://noetixrobotics.feishu.cn/docx/PP2BdXkSUoQ2VOxctJKcEdHanAg?from=from_copylink).

### Build examples

To build the examples inside this repository:

```bash
./build_release.sh
```

### Notice
For more reference information, please go to [N2 DDS_SDK](https://noetixrobotics.feishu.cn/docx/IB1Pd2sQdoxPOLxP1ahcTHlfngk).
